Git

  • git ls-files - Выводит список всех файлов, которые в настоящее время отслеживаются Git в репозитории.
  • cd - команда в Git (сокращение от «Change Directory») для перехода в определённый каталог.
  • pwd - расшифровывается как "Present working directory", что переводится как "текущая рабочая директория". То есть команда pwd показывает, в какой сейчас папке мы находимся.
  • mkdir - расшифровывается как "Make directory", что переводится как "создать папку".
  • git mv - позволяет одновременно переименовать или переместить файл и автоматически обновить индекс, чтобы изменения были учтены при следующем коммите.
